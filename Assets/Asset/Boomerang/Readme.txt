# Overview / 概要
This is Unity package for boomerang in cluster.
clusterで遊べるブーメランのUnityパッケージです。
[cluster](https://cluster.mu/w/59e8e4a2-c7a7-4ee3-b52d-7ad824e7f0c6)

# How to Use / 使用方法
1. メニューのAssets/Import Packageからインポートします。
2. インポートしたデータのPrefabsを使用します。
